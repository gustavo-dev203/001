# MVP – Produto Mínimo Viável

## Objetivo do MVP
Criar uma versão funcional do sistema que permita a administração básica
de eventos pela igreja.

## Funcionalidades Essenciais
- Página inicial
- Login simples para administrador
- Criar eventos
- Editar eventos
- Listar eventos
- Banco de dados simples para armazenar eventos e admins
- Estilização mínima e funcional

## Fora do Escopo do MVP
- Calendário visual avançado
- PWA instalável
- Sistema completo de permissões
- Segurança avançada
- APIs externas
- Design refinado
