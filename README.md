# Sistema Web para Igreja

Este projeto é um sistema web desenvolvido para uma igreja, com foco em
responsividade, simplicidade e controle administrativo.

O objetivo é substituir uma solução atual lenta e limitada, oferecendo
mais autonomia para a igreja na gestão de eventos e informações.

## Tecnologias
- Python (Flask)
- HTML, CSS, JavaScript
- Banco de dados simples (SQLite)
- Arquitetura front-end e back-end integrada

## Status do Projeto
MVP em desenvolvimento

Atualmente o projeto está na fase inicial, com foco na construção das
funcionalidades essenciais antes da expansão para recursos avançados
(PWA, segurança avançada, melhorias visuais, etc).
